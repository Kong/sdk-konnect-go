# PortalMCPRegistryPublications

## Overview

APIs related to MCP registry publications on Konnect Developer Portals.

### Available Operations

* [ListPortalMcpRegistryPublications](#listportalmcpregistrypublications) - List portal MCP registry publications
* [ListPortalMcpRegistries](#listportalmcpregistries) - List MCP Registries
* [ListPortalComputedMcpRegistries](#listportalcomputedmcpregistries) - List computed MCP Registries

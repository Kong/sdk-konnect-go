# Mode

Mode of the checkout session.

Currently only "setup" mode is supported for collecting payment methods.

## Example Usage

```go
import (
	"github.com/Kong/sdk-konnect-go/models/metering"
)

value := metering.ModeSetup
```


## Values

| Name        | Value       |
| ----------- | ----------- |
| `ModeSetup` | setup       |

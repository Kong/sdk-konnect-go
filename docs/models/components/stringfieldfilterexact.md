# StringFieldFilterExact

| `Oeq`                                         | **string*                                     | :heavy_minus_sign:                            | The field matches any of the provided values. |
| `Neq`                                         | **string*                                     | :heavy_minus_sign:                            | The field does not match the provided value.  |

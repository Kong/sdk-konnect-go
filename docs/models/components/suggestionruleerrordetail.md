The type of error that occured when evaluating suggestion rule configuration against given integration record.<br/>  - `error`: represents a generic error.<br/>  - `timeout`: timeout occured when evaluating jq expression<br/>  - `invalid-expression`: jq expression could not be evaluated due to syntax error(s).<br/>  - `result-not-boolean`: jq expression was not evaluated to a boolean<br/>  - `result-not-string`: jq expression was not evaluated to a string<br/>  - `unexpected-result`: jq expression was not evaluated to a result that could not be determined<br/>  - `InvalidLabelKeyInputType`: Indicates the label key is of an invalid type (i.e. number or symbol)<br/>  - `InvalidEmptyLabelKey`: Indicates the label key is empty<br/>  - `InvalidLabelKeyFormat`: Indicates that the format of the label key is invalid, ex: "-bad-format-"<br/>  - `LabelKeyHasReservedPrefix`: Indicates that the label key starts with a Kong-reserved key prefix, ex: "kong-test-key"<br/>  - `MaxLabelKeyCharsExceeded`: Indicates that a label key exceeds the maximum number of allowed characters<br/>  - `InvalidLabelValueInputType`: Indicates the label value is of an invalid type (i.e. number or symbol)<br/>  - `InvalidEmptyLabelValue`: Indicates that the label value is empty<br/>  - `InvalidLabelValueFormat`: Indicates that the format of the label value is invalid, ex: "-bad-format-"<br/>  - `MaxLabelValueCharsExceeded`: Indicates that a label value exceeds the maximum number of allowed characters<br/>  - `InvalidLabelsInputType`: Indicates that a collection of labels is of an invalid type (i.e. array instead of key-value object)<br/>  - `MaxLabelsPerEntityExceeded`: Indicates that a collection of labels exceeds the maximum number of labels allowed for a given entity<br/>  - `ConflictingKeys`: Indicates that labels keys are not unique<br/>  - `InvalidName`: Indicates that the name is invalid<br/>  - `InvalidDisplayName`: Indicates that the display name is invalid<br/>  - `InvalidDescription`: Indicates that the description is invalid<br/>  - `InvalidLabels`: Indicates that the labels are invalid<br/>  - `InvalidMetadata`: Indicates

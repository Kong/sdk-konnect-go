# HashOn

What to use as hashing input. Using `none` results in a weighted-round-robin scheme with no hashing.


## Values

| Name               | Value              |
| ------------------ | ------------------ |
| `HashOnConsumer`   | consumer           |
| `HashOnCookie`     | cookie             |
| `HashOnHeader`     | header             |
| `HashOnIP`         | ip                 |
| `HashOnNone`       | none               |
| `HashOnPath`       | path               |
| `HashOnQueryArg`   | query_arg          |
| `HashOnURICapture` | uri_capture        |

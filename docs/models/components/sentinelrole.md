# SentinelRole

Sentinel role to use for Redis connections when the `redis` strategy is defined. Defining this value implies using Redis Sentinel.


## Values

| Name                 | Value                |
| -------------------- | -------------------- |
| `SentinelRoleMaster` | master               |
| `SentinelRoleSlave`  | slave                |
| `SentinelRoleAny`    | any                  |
